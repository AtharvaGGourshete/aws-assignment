# Sports Ecommerce Web App

A full-stack sports equipment ecommerce application with a frontend and backend.

## Getting Started

Follow the steps below to run the project locally.

## Clone the Repository

```
git clone https://github.com/AtharvaGGourshete/aws-assignment.git
cd aws-assignment
```

## Frontend Setup

```
cd frontend
npm install
npm run dev
```

The frontend runs on http://localhost:5173

## Backend Setup

Open a new terminal and run:

```
cd backend
npm install
npm run dev
```

The backend runs on http://localhost:5000

## Project Structure

```
aws-assignment/
│
├── frontend/     # React + Vite
├── backend/      # Node.js + Express
```

## Requirements

- Node.js (v18 or higher)
- npm or yarn
- Git

## Notes

- Run frontend and backend in separate terminals  
- Backend should be running before frontend makes API calls  
- Create .env files if required  
- Handle CORS if ports are different  

## Tips

- Change ports in config or .env if already in use
